# 安装

## 使用包管理器

###  NPM
```
npm install element-plus --save
```

###  Yarn
```
yarn add element-plus
```

###  pnpm
```
pnpm install element-plus
```


### cnpm

```
cnpm install element-plus --save
```

###  cnpm 安装 淘宝镜像
```
# cnpm 安装
$ pnpm install element-plus

# 淘宝镜像
$ npm install cnpm -g --registry=https://registry.npmmirror.com
```
